# 📋 Complete System Setup & Data Entry Guide

## 🎯 **System Overview**

Your Harding Homes management system is now complete with all features integrated. This guide will walk you through setting up your data and using all the features.

## 2️⃣ **TEAM MEMBER SETUP**

### **Adding Team Members**
Navigate to: **Team** page

**For Each Team Member:**
1. Click **"Add Team Member"** button
2. Fill in details:
   - Full Name
   - Email Address (they'll receive login credentials)
   - Phone Number
   - Role: Owner / Office Manager / Site Manager / Builder / Subcontractor
   - Employment Type: Full-time / Part-time / Contract
3. Click **"Add Member"**
4. System sends invitation email automatically
5. Share credentials via WhatsApp/SMS if email doesn't arrive

**Default Team Roles:**
- 👑 **Owner**: Full system access, all permissions
- 🏢 **Office Manager**: Manage jobs, customers, quotes, invoices
- 🏗️ **Site Manager**: Assign jobs, track time, manage materials
- 🔨 **Builder**: View assigned jobs, log time, update progress, upload photos
- 🔧 **Subcontractor**: Limited access to specific jobs only

---

## 3️⃣ **FLEET MANAGEMENT SETUP**

### **Adding Vehicles**
Navigate to: **Fleet Management** page

**For Each Vehicle:**
1. Click **"Add Vehicle"** (if available, or use company settings)
2. Enter:
   - Registration Number (e.g., AB12 CDE)
   - Make & Model (e.g., Ford Transit)
   - Year
   - Asset Location (e.g., Main Yard, Workshop)
   - MOT Due Date
   - Service Due Date
   - Current Mileage
   - Assigned To (dropdown of team members)
   - Notes
3. Save vehicle

**Reporting Vehicle Issues:**
- Anyone can report issues
- Click **"Report Issue"** button
- Select vehicle, issue type, priority
- Describe the problem
- Managers will see and resolve

**Booking Van Services:**
- Office Managers can book:
  - Regular Services
  - MOT Appointments
  - Repairs
  - Cleaning Days
  - Organisation Days
- Van automatically marked unavailable during service

## 4️⃣ **INVENTORY & MATERIALS**

### **Adding Materials Stock**
Navigate to: **Inventory & Stock** → **Materials** tab

**For Common Building Materials:**
1. Click **"Add Item"**
2. Enter:
   - Item Name (e.g., "Plasterboard 12.5mm")
   - Category: Timber / Plaster / Cement / Electrical / Plumbing / Fixings / Other
   - Current Quantity
   - Minimum Stock Level (reorder alert threshold)
   - Unit (sheets / bags / meters / boxes / pieces)
   - Unit Cost
   - Supplier Name
   - Supplier Contact
   - Storage Location (e.g., "Unit 1 - Bay A3")
   - Notes
3. Save item

**Stock Level Alerts:**
- 🟢 **In Stock**: Above minimum level
- 🟡 **Low Stock**: At or below minimum level
- 🔴 **Out of Stock**: Zero quantity

---

### **Adding Tools & Equipment**
Navigate to: **Inventory & Stock** → **Tools & Equipment** tab

**For Each Tool:**
1. Click **"Add Item"**
2. Enter:
   - Tool Name (e.g., "Makita Impact Driver")
   - Category: Power Tools / Hand Tools / Safety Equipment / Ladders / Other
   - Asset Location (Lock-up / Van / Workshop / Storage)
   - Serial Number (if applicable)
   - Purchase Date
   - Purchase Cost
   - **Condition Report**: Excellent / Good / Fair / Poor / Needs Repair
   - Current Location/Assigned To
   - Notes
3. Save tool

**Tool Condition Tracking:**
- 🟢 **Excellent/Good**: Ready to use
- 🟡 **Fair**: Working but showing wear
- 🟠 **Poor**: Needs attention soon
- 🔴 **Needs Repair**: Don't use, requires service

**Tool Checkout System:**
- Click **"Assign Tool"** to check out
- Select team member and expected return date
- Tool status changes to "Checked Out"
- Click **"Mark Returned"** when tool comes back
- Update condition if changed

## 6️⃣ **JOB CREATION & MANAGEMENT**

### **Creating a New Job**
Navigate to: **Jobs** page → **"Add Job"**

**Step 1: Basic Information**
- Job Title (e.g., "Kitchen Extension")
- Customer (select from dropdown)
- Job Type: Extension / Loft Conversion / Kitchen / Bathroom / Roofing / General Building / Damp Proofing / Timber Treatment / Other
- Priority: High / Medium / Low
- Status: Planning / In Progress / Completed

**Step 2: Dates & Schedule**
- Start Date
- Estimated End Date
- Estimated Duration (e.g., "6 weeks")

**Step 3: Location**
- Job Address (auto-filled from customer if available)
- Add GPS coordinates if available

**Step 4: Team Assignment**
- Select Lead Builder
- Add additional team members
- Assign subcontractors if needed

**Step 5: Specifications**
- Add detailed job description
- List materials required
- Add any special notes

**Step 6: Financial**
- Estimated Cost (from quote)
- Link to Quote (if created)

---

### **Job Management Workflow**

**Phase 1: Planning (Before Start)**
1. Create job from accepted quote
2. Assign team members
3. Check material availability
4. Schedule start date
5. **Take "Before" photos** of site
6. Status: **Planning**

**Phase 2: Job Starts**
1. Click **"Start Job"** button
2. Status changes to: **In Progress**
3. Customer receives automatic email notification
4. Customer portal access activated
5. Team can now log time and upload progress photos

**Phase 3: During Job**
- Team logs daily hours in **Time Logs** tab
- Upload progress photos regularly
- Add materials used via **Materials** tab
- Track purchase orders
- Communicate with customer via **Communications** tab
- Respond to any customer concerns

**Phase 4: Job Completion**
1. Take final **"After" photos**
2. Upload completion documents:
   - Building Control certificates
   - Electrical certificates
   - Gas Safe certificates
   - Warranties
   - Care instructions
3. Click **"Mark Complete"** button
4. Status changes to: **Completed**
5. Customer receives completion email
6. Generate final invoice

## 🆘 **COMMON TROUBLESHOOTING**

### **Email Invitations Not Sending**
- Check spam/junk folders
- Verify email address is correct
- Check Supabase email settings (see EMAIL_SETUP_GUIDE.md)
- Use manual credential sharing via WhatsApp/SMS as backup

### **Customer Can't Access Portal**
- Verify email address matches customer record
- Check portal access is enabled for the job
- Send password reset link
- Ensure job status is "In Progress" or "Completed"

### **Photos Not Uploading**
- Check file size (max 10MB recommended)
- Check internet connection
- Try different image format (JPG recommended)
- Clear browser cache

### **Maps Not Opening**
- Ensure address is complete and accurate
- Check GPS permissions on mobile device
- Try clicking "Open in Maps" again
- Verify maps app is installed on mobile

### **Time Logs Not Saving**
- Check you're logged in
- Verify hours entered is a valid number
- Ensure date is selected
- Check you have permission for that job
