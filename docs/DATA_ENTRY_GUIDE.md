# 📊 Complete Data Entry Guide for Harding Homes

## 🎯 Quick Start: Adding Your Business Data

### 1️⃣ **Add Customers** (5 minutes each)

**Location:** Dashboard → Customers → "Add Customer" button

**Required Fields:**
- ✅ Full Name (e.g., "John Smith")
- ✅ Email (e.g., "john@example.com")
- ✅ Phone (e.g., "07700 900123")
- ✅ Address (full address for jobs)

**Optional Fields:**
- Company name (for business customers)
- Notes (special requirements, preferences)

**Pro Tip:** Add customers as you convert leads - no need to pre-populate everything!

---

### 2️⃣ **Add Leads** (3 minutes each)

**Location:** Dashboard → Leads → "Add Lead" button

**Required Fields:**
- ✅ Customer Name
- ✅ Source (Checkatrade, Referral, Website, etc.)
- ✅ Service Type (New Build, Extension, Renovation, etc.)
- ✅ Status (New, Contacted, Quote Sent, etc.)

**Best Practice:**
1. Add lead immediately when it comes in
2. Update status as you progress
3. Convert to job when accepted
4. Track conversion rate in Reports

**Sources to Track:**
- Checkatrade
- Word of Mouth
- Website
- Facebook
- Instagram
- Previous Customer

---

### 3️⃣ **Create Jobs** (10 minutes each)

**Location:** Dashboard → Jobs → "Add Job" button

**Required Fields:**
- ✅ Job Title (e.g., "Kitchen Extension - Smith Residence")
- ✅ Customer (select from dropdown)
- ✅ Property Address
- ✅ Start Date
- ✅ Status (Scheduled, In Progress, Completed)

**Optional But Recommended:**
- End Date (estimated completion)
- Assigned Team Members
- Budget/Quote Amount
- Job Description
- Special Instructions

**Job Workflow:**
1. Create job from accepted quote
2. Assign team members
3. Add job sheet and plans
4. Track progress daily
5. Upload photos
6. Mark complete
7. Send customer portal access

---

### 4️⃣ **Upload Job Documents** (2 minutes)

**Location:** Jobs → [Select Job] → Documents Section

**What to Upload:**
- 📋 Job Sheets (PDF/Images)
- 📐 Plans and Drawings
- 📝 Specifications
- 📄 Permits/Approvals
- 📸 Progress Photos

**File Types Supported:**
- PDF, JPG, PNG, HEIC
- Max 10MB per file
- Unlimited files per job

---

### 5️⃣ **Track Materials & Purchases** (5 minutes)

**Location:** Dashboard → Inventory

**For Each Purchase:**
- ✅ Item Name
- ✅ Supplier
- ✅ Cost
- ✅ Job Allocation
- ✅ Purchase Order Number
- ✅ Date

**Benefits:**
- See total material cost per job
- Track supplier spending
- Identify cost overruns early
- Monthly financial reports

---

### 6️⃣ **Log Work Hours** (Daily, 1 minute)

**Location:** Jobs → [Select Job] → Hours Tracking

**Each Team Member Logs:**
- Date worked
- Hours worked
- Tasks completed
- Any issues

**Admin Benefits:**
- Accurate payroll data
- Job profitability tracking
- Team productivity insights
- Monthly analytics

## 🔄 **Importing from TradeGrow**

### **Step 1: Export from TradeGrow**

1. Log into TradeGrow
2. Go to each section (Customers, Jobs, etc.)
3. Click "Export" or "Download CSV"
4. Save files to your computer

### **Step 2: Prepare Import Files**

**Format your CSV files with these columns:**

```
name,email,phone,address,notes
```

```
title,customer_email,address,start_date,status,budget,description
```

## 💡 **Pro Tips for Fast Data Entry**

### **Time-Savers:**
1. **Tab Navigation** - Use Tab key to jump between fields
2. **Copy/Paste** - For similar jobs, copy previous entries
3. **Batch Entry** - Add all customers in one session
4. **Mobile Entry** - Builders add data on-site
5. **Voice Dictation** - Use phone voice-to-text for notes

### **Don't Overthink:**
1. **Start Small** - Active jobs only at first
2. **Add as You Go** - Historical data comes later
3. **Minimum Viable** - Basic info beats perfect info never entered
4. **Team Helps** - Delegate data entry to relevant people

### **Quality Over Quantity:**
1. **Accurate Addresses** - Critical for mapping
2. **Complete Contact Info** - Phone + Email minimum
3. **Clear Job Descriptions** - Helps team understand scope
4. **Regular Updates** - Daily is better than perfect

## 🆘 **Common Questions**

**Q: Do I need to add everything from TradeGrow?**
A: No! Start with active jobs and customers. Historical data is optional.

**Q: Can I bulk import customers?**
A: I can build a CSV import tool if you have 50+ customers. For smaller numbers, manual is faster.

**Q: What if I make a mistake?**
A: Everything is editable! Just click the job/customer and update.

**Q: Can team members add data?**
A: Yes! Builders can add photos/hours, office staff can add leads/jobs.

**Q: How do I avoid duplicate customers?**
A: System checks email/phone - you'll get a warning if duplicate detected.
